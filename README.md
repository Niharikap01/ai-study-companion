# 🌌 MyGenius – AI Study Companion

🚀 **MyGenius** is a next-generation **AI-powered study companion** built using Flutter, designed with a stunning **galaxy-themed UI** to make learning immersive, interactive, and engaging.

---

## ✨ Key Features

### 🧠 AI Summary (UI Ready)

* Input text or images
* Generates structured summaries
* Beautiful dual-panel layout with future diagram visualization

### 📚 Flashcards Galaxy

* Interactive flashcards in a **cosmic orbit UI**
* Smooth animations for engaging revision
* Clean and distraction-free design

### 📅 Smart Study Planner

* Generates personalized study plans
* Designed to optimize productivity
* Future support for reminders and scheduling

### 🗺 Interactive Concept Maps

* Input any topic and visualize it as a **dynamic concept map**
* Galaxy-themed structure for better understanding

### 📊 Progress Universe (Gamified)

* Tracks learning progress in a **game-like space UI**
* Displays:

  * Study streaks
  * Completed topics
  * Performance insights

### 👤 Profile System (UI)

* Dedicated profile screen
* Clean and minimal layout
